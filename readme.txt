Project Dungeon

Bienvenue dans le fichier README du projet "Project Dungeon". Ce document vise à fournir une vue d'ensemble des contributions de chaque membre de l'équipe.

Tâches du Projet:
------------------

1. Création de la classe "stairs" :
   - Responsable : Abdoulbastoi
   - Description : Abdoulbastoi a créé une classe "stairs" qui permet de déclencher le changement de niveau lorsque le héros atteint les escaliers.

2. Architecture des niveaux et placement des tuiles :
   - Responsable : Rayan
   - Description : Rayan a conçu l'architecture des niveaux du jeu et a placé les tuiles pour créer un environnement de jeu cohérent et engageant.

3. Implémentation de la capacité de course du héros :
   - Responsable : Amine
   - Description : Amine a intégré la fonctionnalité permettant au héros de courir ou de marcher en fonction des conditions du jeu.
